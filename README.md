<h1>BMI Test</h1>

a simple projects that takes a few infromations from a submitted form in **https://osamabozalim.me/bmi-test** and performs a calculation to return your body type.

## Available Body Types:
  * UnderWeight
  * Normal Range
  * Overweight
  * Obesity
  * Severe Obesity
  
 <h1>Technologies:</h1>
 
  * `HTML/CSS`
  * `JavaScript`
